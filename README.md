# Expense Data Analysis Project

## Objective
To analyze personal/business expenses and identify spending patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## Dataset
The dataset contains expense date, category, amount, and payment mode.

## Analysis Performed
- Monthly expense analysis
- Category-wise spending
- Highest expense category
- Visualization of expenses

## Conclusion
This project helps understand spending behavior and control expenses.
